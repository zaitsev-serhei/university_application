# university_application

Spring Boot Console Application

# SETUP 
Please, make sure Postgres Server is running on your local machine. Update port in properties file if needed

Make sure to create DB and User in advance.

If additional logs needed, please update the properties file respectively

# RUN
use command line or IDEA tools to run the application

mvn spring-boot:run (for Windows cmd/ mvn should be installed and added to Path)
